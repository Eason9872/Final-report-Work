粒子效果（tsParticles）

在這個網頁裡，我加入了一個我覺得很有趣的效果，就是背景會有很多小粒子漂浮移動，看起來很有科技感，而且滑鼠靠近時還會互動。這個功能是我用一個叫 tsParticles 的 JavaScript 套件做出來的。

這是什麼？

tsParticles 是一個可以在網頁上產生「粒子動畫」的工具，它其實是用 HTML 的 <canvas> 畫布，再搭配 JavaScript 去畫出一顆一顆小圓點，並讓它們移動、連線或對滑鼠有互動。

怎麼做的？

先引入 CDN、放一個粒子容器，再用 JavaScript 設定參數就可以。例如我設定了：

  • 顆粒數量（我用的是 80 顆）
  • 粒子顏色和大小
  • 移動速度

為什麼想加這個？

一開始是想讓首頁看起來不要太單調， 跑去詢問gpt 該怎麼讓網頁更好看 ai就推薦給我一些選擇 我會想選擇粒子效果 是因為我覺得加個動畫效果讓它有「活的感覺」 我覺得很好看 很帥，後來研究這個套件後發現真的蠻好玩，也讓整個網頁有一點未來感。 更重要的是，我也藉這個機會學習怎麼使用外部 JavaScript 套件， 還能順便了解一點 canvas 的原理，對我來說很有收穫。

心得

一開始接觸 HTML 的時候真的覺得很難，不知道每個標籤是做什麼的，也常常搞不懂為什麼網頁長得怪怪的。
但在慢慢摸索、練習的過程中，我開始發現 HTML 就像是在搭積木，一點一滴把自己的想法變成畫面，感覺超有成就感。
從一開始的挫折，到後來覺得「Damn，原來是這樣啊」這種轉變讓我對網頁設計越來越有興趣，也更想繼續學下去！

背景圖重疊

以下圖片使用了背景圖重疊background-repeat 這個程式碼

background-repeat 是一個 CSS 屬性，用來控制背景圖片是否要「重複」顯示（平鋪），以及怎麼重複。

🖼️ 範例

body {
background-image: url('bg.png');
background-repeat: no-repeat;
}
👉 背景圖片只會出現一次，不會重複。

網頁怎麼做的

這個網頁是用純 HTML、CSS 和 JavaScript 寫成的，沒有使用任何框架。

主要使用了以下技術：
  • 響應式設計 (Responsive Design)，讓頁面能在手機、平板和電腦上自動調整排版。
  • 使用 JavaScript 控制選單的開關和內容區塊切換，實現單頁應用效果。
  • CSS 漸變背景和動畫讓頁面看起來更有活力和現代感。
  • 分離 HTML、CSS 和 JavaScript 三個檔案，方便維護和擴充。

希望這個作品可以展現我對基礎網頁設計的學習成果！

學習歷程與想法

這次做的 RWD 網頁，我覺得可以算是我目前最成功的作品，有時候都會回去看一下 在那邊自嗨。

一開始真的滿頭問號，CSS 怎麼寫都跑版、JavaScript 一直寫錯，連選單都不聽話。但慢慢看老師的簡報，一頁一頁跟著試，漸漸就有感覺了。有時候卡住太久，還會詢問GPT 幫忙解救我，真的很香 香噴噴的ai幫忙（ 感謝 AI 🙏）。

做到能讓網頁在電腦、平板、手機都順順的運作，對我來說超有成就感，尤其是看到自己的漢堡選單真的打開來的那一刻（明明只是三條線 😆），我內心是站起來鼓掌的。

雖然還不是什麼超專業的作品，但至少從頭到尾都是我自己做的，這份成品對我來說很有意義，也讓我更確定自己真的喜歡搞這些東西。
