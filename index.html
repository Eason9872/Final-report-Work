背景圖重疊

以下圖片使用了背景圖重疊background-repeat 這個程式碼

background-repeat 是一個 CSS 屬性，用來控制背景圖片是否要「重複」顯示（平鋪），以及怎麼重複。

🖼️ 範例

body {
background-image: url('bg.png');
background-repeat: no-repeat;
}
👉 背景圖片只會出現一次，不會重複。

RWD選單

上方選單特色
採用 漢堡選單（Hamburger Menu） 設計，常見於手機版網站

使用者在手機裝置上會看到三條橫線的按鈕，點擊後展開導覽連結

當畫面變寬，選單會自動顯示成橫列的標題，方便桌機使用者瀏覽

學到什麼？

如何製作行動裝置友善的選單
如何處理文字過長、圖片排版等 RWD 問題
如何用簡單的 JS 和 CSS 提升網頁互動性與易用性

網頁怎麼做的

這個網頁是用純 HTML、CSS 和 JavaScript 寫成的，沒有使用任何框架。

主要使用了以下技術：
  • 響應式設計 (Responsive Design)，讓頁面能在手機、平板和電腦上自動調整排版。
  • 使用 JavaScript 控制選單的開關和內容區塊切換，實現單頁應用效果。
  • CSS 漸變背景和動畫讓頁面看起來更有活力和現代感。
  • 分離 HTML、CSS 和 JavaScript 三個檔案，方便維護和擴充。

希望這個作品可以展現我對基礎網頁設計的學習成果！
